# Registration Form Validation

A simple and responsive Registration Form built using **HTML, CSS, and JavaScript**. This project demonstrates client-side form validation with real-time input checking and error handling.

## 🚀 Features

- ✅ Username validation (cannot be empty)
- ✅ Email format validation using Regular Expression (Regex)
- ✅ Password validation (minimum 6 characters)
- ✅ Real-time validation while typing
- ✅ Displays error messages for invalid inputs
- ✅ Green border for valid fields
- ✅ Red border for invalid fields
- ✅ Submit button remains disabled until all fields are valid
- ✅ Responsive and clean UI

## 📋 Validation Rules

### Username
- Cannot be empty.

### Email
- Must follow a valid email format.
- Example:
  - ✔ example@gmail.com
  - ❌ examplegmail.com

### Password
- Cannot be empty.
- Must contain at least **6 characters**.
